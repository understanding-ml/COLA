## Prerequisites

Before starting these tutorials, ensure you have:
- Python 3.7+ installed
- COLA installed (`pip install xai-cola`)
- Basic understanding of machine learning
- Familiarity with pandas and scikit-learn

## Tutorial Format

Each tutorial includes:
- **Learning objectives** - What you'll learn
- **Prerequisites** - What you need to know first
- **Step-by-step guide** - Detailed instructions with code
- **Complete example** - Full working code
- **Exercises** - Practice problems
- **Solutions** - Exercise solutions

## Running Tutorials

All tutorials can be run as:

1. **Jupyter Notebooks**: Download from `examples/` directory
2. **Python Scripts**: Copy code blocks and run in your environment
3. **Interactive**: Follow along in your own IDE
