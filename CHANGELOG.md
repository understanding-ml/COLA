# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added
- Initial release of COLA (COunterfactual explanations with Limited Actions)
- Core COLA functionality for refining counterfactual explanations
- Support for multiple counterfactual explainers (DiCE, DisCount, Alibi-CFI, ARecourseS)
- Support for multiple matching policies (OT, ECT, NN, CEM)
- Feature attribution methods (PSHAP)
- Visualization tools for heatmaps and highlighting changes
- Built-in datasets (GermanCredit, Compas, HELOC, HotelBookings)
- Support for Pandas and Numpy data interfaces
- Support for scikit-learn and PyTorch models
